# task 4
## ls.c
выводит файлы текущего каталога. Флаги: -R (рекурсивный обход подкаталогов), -l (подробный вывод атрибутов), -g (показ группы владельца). Использует системные вызовы stat(), lstat(), readdir() для получения информации о файлах. Алгоритм: чтение записей каталога, пропуск . и .., динамическое выделение памяти для хранения имён, вывод списка, рекурсивный обход подкаталогов при флаге -R.
## grep.c
Поиск строк в файле по подстроке. Идея: программа анализирует аргументы, проверяет наличие флага -v для инвертированного поиска, открывает указанный файл и построчно считывает его. Для каждой строки с помощью функции strstr проверяется наличие заданной подстроки, и в зависимости от флага -v выводятся либо содержащие, либо не содержащие подстроку строки.

## pwd.c
Вывод абсолютного пути текущей рабочей директории. Идея: программа сначала определяет максимально возможную длину пути через системный вызов pathconf, выделяет под него динамический буфер, затем получает сам путь с помощью getcwd и выводит его в стандартный поток вывода. Если getcwd возвращает ошибку, программа выводит соответствующее сообщение.

## ln.c 
Создание жестких и символических ссылок между файлами. Идея: программа разбирает аргументы командной строки: если передан флаг -s, создается символическая ссылка с помощью symlink, иначе — жесткая ссылка через link. После успешного создания выводится сообщение о типе созданной ссылки и ее направлении.

## cat.c
Вывод содержимого одного или нескольких файлов на экран. Идея: программа обрабатывает аргументы командной строки, запоминая флаг -n для нумерации строк и собирая имена файлов в динамический массив. Затем для каждого файла она открывает его, построчно считывает содержимое через fgets и выводит либо с нумерацией строк, либо без нее, в зависимости от наличия флага -n.
