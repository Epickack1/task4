# task 4
## ls.c
ls — выводит файлы текущего каталога. Флаги: -R (рекурсивный обход подкаталогов), -l (подробный вывод атрибутов), -g (показ группы владельца). Использует системные вызовы stat(), lstat(), readdir() для получения информации о файлах. Алгоритм: чтение записей каталога, пропуск . и .., динамическое выделение памяти для хранения имён, вывод списка, рекурсивный обход подкаталогов при флаге -R.
